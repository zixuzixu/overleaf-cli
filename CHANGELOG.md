# Changelog

All notable changes to this project will be documented in this file.

## [0.1.1] - 2026-04-13

### Added
- Negation patterns in `.overleafignore` (prefix with `!` to un-ignore files). Lets you keep PDF figures like `figures/*.pdf` while still ignoring compiled PDFs.
- Documentation for negation pattern syntax with examples.

## [0.1.0] - 2026-04-06

### Added
- Initial release: login, clone, pull, push, init, create commands.
- Cookie-based and git-bridge authentication.
- `.overleafignore` support with default LaTeX compile artifact patterns.
