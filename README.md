# overleaf-cli

A Python CLI tool for bidirectional sync with Overleaf. Login, pull, push, and manage your Overleaf projects from the terminal.

## Features (planned)

- **Login**: Authenticate via browser cookie extraction or manual paste
- **Projects**: List your Overleaf projects
- **Clone**: Download a project to local directory
- **Pull**: Pull remote changes from Overleaf
- **Push**: Push local changes to Overleaf
- **Status**: Show local/remote diff
